# 🧠 Beecrowd - Prática de Lógica em Java

Este repositório foi criado com o objetivo de praticar lógica de programação
utilizando problemas da plataforma Beecrowd.

Também utilizo este projeto para me acostumar com o uso do Git e versionamento
de código no dia a dia.

## 📌 Objetivos

- Melhorar lógica de programação
- Praticar Java
- Criar consistência nos estudos
- Aprender Git na prática

## 📂 Organização

Os exercícios serão organizados por categorias do Beecrowd.
